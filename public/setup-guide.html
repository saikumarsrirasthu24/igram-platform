AgriUber — Google Sheets Setup

Connect your booking forms to Google Sheets in 15 minutes. No server needed.

Free forever No server required 3 sheets to create

What you will create:

3 Google Sheets — one for Bookings, one for Farmer Registrations, one for Machine Owner Registrations. Each sheet connects to the website via a Google Apps Script Web App URL.

PART 1 — Create the Booking Sheet

1

Create the Google Sheet

Go to sheets.google.com → click + to create new sheet.

Name it: AgriUber — Bookings

Add these column headers in Row 1:

A: Timestamp
B: Farmer Name
C: Phone
D: Village
E: Taluka
F: Equipment
G: Booking Date
H: Land Size
I: Crop Type
J: Notes
K: Language
L: Source
2

Open Apps Script

In your Google Sheet: click Extensions → Apps Script

Delete all existing code. Paste the code below:

function doPost(e) {
  try {
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    var data  = e.parameter;

    sheet.appendRow([
      data.timestamp   || new Date().toLocaleString('en-IN'),
      data.name        || '',
      data.phone       || '',
      data.village     || '',
      data.taluka      || '',
      data.equipment   || '',
      data.date        || '',
      data.land        || '',
      data.crop        || '',
      data.notes       || '',
      data.language    || '',
      data.source      || 'AgriUber Website'
    ]);

    return ContentService
      .createTextOutput(JSON.stringify({ result: 'success' }))
      .setMimeType(ContentService.MimeType.JSON);

  } catch(err) {
    return ContentService
      .createTextOutput(JSON.stringify({ result: 'error', message: err.message }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

// Test function - run this once to verify it works
function doGet(e) {
  return ContentService
    .createTextOutput('AgriUber Booking Sheet is working!')
    .setMimeType(ContentService.MimeType.TEXT);
}
3

Deploy as Web App

Click Deploy → New deployment

Set these options:

Type: Web App
Execute as: Me (your Google account)
Who has access: Anyone ← Very important!

Click Deploy → Authorize access → Choose your Google account → Allow.

📋 COPY the Web App URL — it looks like:

https://script.google.com/macros/s/AKfycb.../exec
4

Paste URL into agriuber.html

Open agriuber.html in any text editor. Find this line:

const BOOKING_SHEET_URL  = 'YOUR_BOOKING_SHEET_WEBAPP_URL';

Replace YOUR_BOOKING_SHEET_WEBAPP_URL with the URL you copied. Result:

const BOOKING_SHEET_URL  = 'https://script.google.com/macros/s/AKfycb.../exec';

8

Set agriuber.html as default AgriUber page

If your hosting supports URL redirects, add this in your .htaccess:

RewriteRule ^agriuber/?$ /agriuber.html [L]
RewriteRule ^agriuber-booking/?$ /agriuber.html#booking-form [L]

PART 5 — Notify yourself on new bookings

9

Get email alert for every new booking

In your Booking Google Sheet: click Tools → Notification rules

Select: Any changes are made → Email immediately.

You'll get an email every time a new booking is submitted. No manual checking needed.
